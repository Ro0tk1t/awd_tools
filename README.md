# awd_tools
4 AWD competition

## install

```bash
sudo apt install masscan
sudo chmod u+s /usr/bin/masscan
make install
```


## Quick start

*use tools directly*

see command help:
```bash
tony --help
```
start scan:
```bash
tony --enemise 127.0.0.1 10.0.8.202/30 --change --oldpwd 123456 abcdef --newpwd hydra --usernames root ubuntu scanv 
```
